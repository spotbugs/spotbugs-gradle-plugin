Package com.github.spotbugs.snom

Package com.github.spotbugs.snom Description

This package provides public classes/interfaces for users. All published classes/interfaces/methods in this package (not including sub-packages) are treated as public and keep compatible.

The SNOM stands for SpotBugs Newly Organized Mechanism.
